# Kristina Ryaboy Portfolio

Personal portfolio site showcasing product design work, case studies, and design philosophy. Built and deployed by Kristina Ryaboy.

**Live site:** [kristinaryaboy.github.io/KRPortfolio](https://kristinaryaboy.github.io/KRPortfolio)

## About

A hand-built portfolio featuring:

- **Homepage** with hero, design philosophy, process, skills, and contact form
- **Arcus** case study, a shipped consumer mobile app live on the iOS App Store and Google Play, partnered with Ace Archers
- **Discover Movement Connect (DMC)** case study, a coaching platform for a functional movement and pain management gym in Dedham, MA
- **Portfolio Website** case study, covering the design and build of this site itself
- **4th & Model** sneak peek card, linking to a sports analytics platform currently in development

## Tech stack

- HTML5, CSS3, JavaScript (vanilla, no frameworks)
- AI-assisted development workflow using Claude Code
- Static site, deployed via GitHub Pages
- No build step required

## Accessibility

Built with accessibility as a craft floor rather than a checklist item:

- Skip links for keyboard navigation
- Descriptive alt text on content images, decorative treatment for purely visual imagery
- Semantic HTML with proper heading hierarchy
- WCAG-aware color contrast and focus indicators
- Comprehensive `prefers-reduced-motion` support
- An in-app motion toggle for users who prefer to disable motion without changing system settings
- ARIA attributes where semantic HTML alone is insufficient (e.g., `aria-hidden` on decorative video)
- Language attribute set on every page

## Design system

- Dark theme with magenta accent (`#E91E79`)
- Typography: Josefin Sans (headings) and Inter (body)
- Custom CSS variables for color, spacing, and timing
- Responsive across desktop, tablet, and mobile breakpoints

## Local development

This is a static site with no build step. To run locally:

```bash
git clone https://github.com/kristinaryaboy/KRPortfolio.git
cd KRPortfolio
# Open index.html in any browser, or serve with a local static server
```
